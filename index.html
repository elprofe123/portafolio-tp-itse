Acerca de Mí

Soy estudiante del Instituto Tecnológico de Santiago del Estero (ITSE), cursando el último año de la Tecnicatura en Desarrollo de Software. Actualmente participo en el desarrollo de un sitio web para la Secretaría de Turismo, aplicando mis conocimientos en Python y Django orientados al desarrollo de soluciones web.

Mi enfoque se centra en escribir código limpio, crear interfaces de usuario intuitivas y resolver problemas complejos de manera eficiente. Siempre estoy buscando aprender nuevas tecnologías y mejorar mis habilidades.

Habilidades Técnicas:

  • React & Next.js
  • Node.js & Express
  • Python & Django
  • HTML5 & CSS3
  • Bootstrap & Tailwind
  • MongoDB & PostgreSQL
  • Git & GitHub
